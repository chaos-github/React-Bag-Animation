# Bag Animation UI

一个用于显示 APNG 动画的 React 组件，支持全屏遮罩和交互式拖拽。

## 开发

### 安装依赖

```bash
npm install
```

### 开发模式（预览组件效果）

```bash
npm run demo
```

这会在 `http://localhost:3000` 启动开发服务器，你可以在浏览器中实时预览组件效果。

### 构建组件

```bash
npm run build
```

### 构建演示页面

```bash
npm run demo:build
```
